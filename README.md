 短剧下载神器开源版介绍

## 项目定位

短剧下载神器开源版是一个 Windows 本地短剧搜索与下载辅助工具，提供桌面端和浏览器端两种使用方式。项目面向个人学习、资料整理和开源二次开发，代码结构尽量保持简单，便于阅读、修改和重新打包。

## 版本说明

### 桌面版

启动文件：`短剧下载神器桌面版.exe`

桌面版使用 Windows 原生窗口界面。界面包含全网搜索、下载任务和设置三个主要页面，适合日常直接使用。

### 浏览器版

启动文件：`短剧下载神器开源版.exe`

浏览器版启动后会在本机运行服务，并打开 Web 界面。前端页面适合继续扩展，也方便改造成在线管理界面或局域网工具。

## 主要功能

- 关键词搜索短剧
- 页码选择
- 数据来源选择
- 分类关键字过滤
- 搜索结果表格展示
- 批量加入下载任务
- 手动粘贴短剧 ID 加入队列
- 导出搜索结果 CSV
- 导出下载链接 CSV
- 本机配置保存
- 自定义背景墙和应用图标

## 表格字段

搜索结果包含以下字段：

- 序号
- 作者
- 剧名
- 短剧 ID
- 集数
- 时长
- 上线时间
- 分类
- 来源
- 详情链接

## 已接入来源

当前版本接入或预留了以下公开来源：

- 红果短剧
- 红果漫剧
- 爱奇艺短剧
- FlexTV
- 熊猫短剧
- 趣看看短剧
- 全网聚合公开检索

不同平台公开页面提供的数据字段不完全一致。如果平台没有公开上线时间、总时长或可直接下载 ID，程序会显示可获取的信息，并保留详情链接。

## 源码结构

```text

static/index.html              浏览器版前端页面
static/assets/wallpaper.jpg    背景墙图片
static/assets/app_icon.ico     应用图标
requirements.txt               Python 依赖
短剧下载神器开源版.spec          浏览器版 PyInstaller 配置
短剧下载神器桌面版.spec          桌面版 PyInstaller 配置
```

## 开发环境

建议使用 Python 3.11。

安装依赖：

```powershell
py -3.11 -m pip install -r requirements.txt
```

运行浏览器版源码：

```powershell
py -3.11 app.py
```

运行桌面版源码：

```powershell
py -3.11 desktop_app.py
```

## 打包命令

浏览器版：

```powershell
py -3.11 -m PyInstaller --clean --noconfirm 短剧下载神器开源版.spec
```

桌面版：

```powershell
py -3.11 -m PyInstaller --clean --noconfirm 短剧下载神器桌面版.spec
```

## 配置说明

下载接口可能需要本机配置文件提供设备参数。配置可以在界面中填写并保存，也可以参考 `.env.example` 或程序生成的配置文件进行调整。
