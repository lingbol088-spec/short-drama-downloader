# 短剧下载工具（开源版）

这是一个本地 EXE/Web UI 版短剧下载工具。双击 EXE 后启动本地 Flask 服务，并自动打开浏览器前端。

## 当前界面模块

### 1. 全网搜索界面

包含：

- 搜索关键词
- 分页
- 数据来源选择，例如：红果漫剧、全网聚合、本地导入
- 开始搜索
- 下载选中
- 分类过滤，多个关键字用逗号分隔
- 导出数据
- 表格列：序号、作者、剧名、短剧ID、集数、时长、操作、上线时间、分类
- 底部操作栏：下载选中、导出数据、分类、添加示例行

> 说明：开源版已预留 `/api/search` 搜索适配器入口，默认不内置私有搜索源。后续接入真实搜索源时，只需要在 `app.py` 的 `search_short_drama()` 返回统一字段即可。

### 2. 批量下载和下载界面

包含：

- 批量粘贴短剧ID / video_id
- 加入队列
- 开始下载
- 暂停
- 清空任务
- 下载任务列表
- 单任务状态：等待、下载中、完成、失败
- 完成后支持打开链接、复制链接
- 失败任务支持重试

批量下载会逐个调用后端：

```text
/hg?vid=<短剧ID或video_id>
```

### 3. 设置界面

用于保存本机配置：

- `device_id`
- `install_id`
- `platform`

配置保存到运行目录的 `config.json`，不会写入源码。`config.json` 已加入 `.gitignore`。

## 运行源码

```powershell
cd "D:\zm\短剧下载神器1"
py -3.11 -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

```text

```

## 开源前不要提交

- `build/`
- `dist/`
- `src/`
- `.env`
- `config.json`
- 真实 `device_id` / `install_id` / Cookie / Token

## 搜索源与链接处理

已接入的公开真实搜索源/入口：

- 红果短剧官网：解析 `https://hongguoduanju.com/category`，可返回 `series_id`、剧名、集数、分类、详情链接。
- 红果漫剧官网：公开入口 `https://yuyuedushu.com/`，官网当前未暴露剧目列表时返回入口链接。
- 爱奇艺短剧、FlexTV、熊猫短剧、趣看看短剧：接入公开网页/公开检索 fallback，返回平台入口或公开结果链接。
- 全网聚合：红果官网结果 + 公开网页检索结果。

链接字段：

- `source_url`：搜索结果详情页/平台官网链接，搜索结果导出时会保留。
- `download_url`：下载任务解析完成后的真实下载链接，任务列表会显示“打开/复制链接”，并可通过“导出下载链接”导出 CSV。
- 下载任务保存在浏览器 `localStorage` 的 `duanju_download_tasks` 中，刷新页面不会丢失已解析出的下载链接。

## 桌面版 EXE（不打开浏览器）

已新增真正的 Windows 本地窗口程序：

```text
D:\zm\短剧下载神器1\dist\短剧下载神器桌面版\短剧下载神器桌面版.exe
```

特点：

- 不启动浏览器前端。
- 不需要访问 `127.0.0.1:5000`。
- 直接打开 Windows 桌面窗口。
- 包含：全网搜索、下载任务、设置三个标签页。
- 搜索表格包含：作者、剧名、短剧ID、集数、时长、上线时间、分类、来源、详情链接。
- 下载任务保存到 EXE 同目录 `desktop_tasks.json`，下载链接不会因关闭程序丢失。
- 红果官网未公开上线时间时显示“官网未公开”。
- 红果官网未公开总时长时，按集数给出估算时长，例如“约 116-232 分钟（按每集 1-2 分钟估算）”。
